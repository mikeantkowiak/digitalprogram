Hive Media Group

Influencer Dashboard

Project Overview

Hive Media Group was looking for an app where they could visualize and present data to both the influencers and the account managers. The main capabilities of the dashboard were to allow influencers to view post data in a intuitive user experience. An integrated CRM feature for account managers was implemented to view associated influencers and statistic. The managers could generated statistical reports, and revenue earning reports.

Technical Environment

  • Backend: AWS, Ubuntu, Laravel Forge, Laravel 5

  • Front-end: Design and Mockups, Custom SCSS and Bootstrap 3, gulp for compiling

  • Javascript: Custom integrations of DataTables.js and Chart.js

  • Version Control: BitBucket

Living Style Guide

Aftr the initial designs were created and confirmed, the HTML, CSS and base js frontend frameworks were coded, customized and placed on a static website and kept in a independent git repository. The goal was to separate the design frontend framework from the backend code for easier updated to overall branding, ui elements, and reusable components.

Technical Environment

  • SCSS Framework: Bootstrap 3

  • Front-end: HTML, Datatables.js, Chart.js
