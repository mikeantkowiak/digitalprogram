    Hive Media Group

    Influencer Dashboard

    Hive Media Group worked with many different influencers and this dashboard was created to collect and present data to both the influencers and the account managers. The main capabilities of the dashboard were to allow influencers to view post data in a intuitive user experience. It was also created as a CRM for account managers to view they're associated influencers and statistic. The managers had the ability to generated statistical reports, and revenue earning reports.

    Technical Environment

    Laravel, PHP, Javascript ( chart.js, datatables.js ), Bootstrap 3.0, Database Management, Git, Laravel Forge,
